# 飞书多维表格 AI 助手

一款浏览器插件，专为**飞书多维表格的搭建者**设计。无需任何编程基础，帮助你用 AI 理解自己搭建的表格逻辑，并获得针对性的搭建优化建议。

## ✨ 核心功能

### 📊 一键同步
打开你的飞书多维表格页面后，点击「一键同步」即可自动获取当前表格的所有字段定义、表间关联关系以及自动化流程，让你对整张表的结构一目了然。

### 🤖 AI 协作
基于同步到的表格结构数据，直接和 AI 对话。AI 会结合你的真实表格情况，帮你分析问题、给出搭建优化建议，像一个随时在线的搭建顾问。

> 支持多种 AI 模型（OpenAI、DeepSeek、豆包、通义千问、Kimi 等），你可以按需选择。

---

## 🚀 安装指南

本插件无需上架应用商店，只需通过浏览器的"开发者模式"加载即可使用。

### 第一步：下载代码

点击本仓库右上角的 **Code → Download ZIP**，将下载的压缩包解压到本地任意位置。

### 第二步：在浏览器中加载插件

#### Google Chrome 浏览器
1. 在地址栏输入 `chrome://extensions/` 并回车。
2. 打开右上角的 **「开发者模式」** 开关。
3. 点击左上角的 **「加载已解压的扩展程序」**。
4. 选择你刚才解压出来的文件夹。
5. 完成！你会在浏览器右上角看到插件图标。

#### Microsoft Edge 浏览器
1. 在地址栏输入 `edge://extensions/` 并回车。
2. 打开左下角的 **「开发人员模式」** 开关。
3. 点击 **「加载解压缩」** 按钮。
4. 选择你刚才解压出来的文件夹。
5. 完成！你会在浏览器右上角看到插件图标。

#### 其他 Chromium 内核浏览器（Brave / Arc / Vivaldi 等）
操作步骤与 Chrome 基本一致：进入浏览器的扩展程序管理页面 → 开启开发者模式 → 加载已解压的文件夹。

> ⚠️ **注意**：本插件目前仅支持基于 Chromium 内核的浏览器，暂不支持 Firefox 和 Safari。

---

## ⚙️ 配置 AI

安装后需要配置你自己的 AI 服务密钥，插件本身**不内置任何密钥**。

1. 点击浏览器工具栏的插件图标，打开侧边栏。
2. 点击底部的 **「设置」** 选项卡。
3. 选择你的 AI 提供商（如 DeepSeek），并输入你的 API Key。
4. 保存后即可开始使用 AI 协作功能。

> 💡 不清楚如何获取 API Key？大部分 AI 服务商（如 DeepSeek）注册后即可在其官网的「API 管理」中创建密钥。
